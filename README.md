# 🗒️ To Do List-API
💻 Criação de uma API com Nodejs, para um To-do list

## 👻 Entidades

- Usuários
- Tarefas

## 💻 Como executar o Projeto

### 📌 Instalando as dependências do projeto

         `npm install`

### 📌 Executando o projeto

         `npm run dev`

## 🛣️ Rotas possíveis:

        GET;

        POST;

        PUT;

        DELETE;
